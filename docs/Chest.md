# Chest

Класс `Chest` описывает сундук в игре. Сундук может быть открыт или закрыт, а
внутри него хранится список ID зелий, которые игрок получает при открытии.

## Поля

- `int id` - уникальный номер сундука.
- `bool opened` - состояние сундука. Если значение `true`, сундук уже открыт.
- `std::vector<int> potionIds` - список ID зелий, которые лежат в сундуке.

## Методы

### `Chest()`

Конструктор по умолчанию.

Создаёт сундук с ID `0` и состоянием `opened = false`.

### `Chest(int id, bool opened)`

Конструктор с параметрами.

Входные данные:

- `id` - ID сундука;
- `opened` - начальное состояние сундука.

Создаёт сундук с указанным ID и состоянием.

### `int getId() const`

Возвращает ID сундука.

### `bool isOpened() const`

Возвращает состояние сундука: открыт он или нет.

### `void openChest()`

Меняет состояние сундука на открытое.

### `void addPotionId(int potionId)`

Входные данные:

- `potionId` - ID зелья, которое нужно добавить в сундук.

Добавляет ID зелья в список предметов сундука.

### `std::vector<int> getPotionIds() const`

Возвращает список ID зелий, которые хранятся в сундуке.
