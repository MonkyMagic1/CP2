# Potion

Класс `Potion` описывает зелье лечения. Зелье хранится в инвентаре игрока и
может быть использовано для восстановления здоровья.

## Поля

- `int id` - уникальный номер зелья.
- `std::string name` - название зелья.
- `int healAmount` - количество здоровья, которое восстанавливает зелье.

## Методы

### `Potion()`

Конструктор по умолчанию.

Создаёт пустое зелье с ID `0`, пустым названием и восстановлением `0`.

### `Potion(int id, const std::string& name, int healAmount)`

Конструктор с параметрами.

Входные данные:

- `id` - ID зелья;
- `name` - название зелья;
- `healAmount` - количество восстанавливаемого здоровья.

Создаёт зелье с указанными параметрами.

### `int getId() const`

Возвращает ID зелья.

### `std::string getName() const`

Возвращает название зелья.

### `int getHealAmount() const`

Возвращает количество здоровья, которое восстанавливает зелье.
