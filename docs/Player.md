# Player

Класс `Player` описывает игрока. Он хранит здоровье персонажа, текущую локацию
и инвентарь с ID зелий.

## Поля

- `int health` - здоровье игрока. Значение ограничивается диапазоном от `0` до `100`.
- `int currentLocationId` - ID локации, в которой сейчас находится игрок.
- `std::vector<int> potionIds` - инвентарь игрока, в котором хранятся ID зелий.

## Методы

### `Player()`

Конструктор по умолчанию.

Создаёт игрока со здоровьем `100` и начальной локацией `1`.

### `int getHealth() const`

Возвращает текущее здоровье игрока.

### `void setHealth(int health)`

Входные данные:

- `health` - новое значение здоровья.

Устанавливает здоровье игрока. Если значение меньше `0`, здоровье становится
`0`. Если значение больше `100`, здоровье становится `100`.

### `int getCurrentLocationId() const`

Возвращает ID текущей локации игрока.

### `void setCurrentLocationId(int locationId)`

Входные данные:

- `locationId` - ID новой текущей локации.

Меняет текущую локацию игрока.

### `void takeDamage(int damage)`

Входные данные:

- `damage` - количество урона.

Уменьшает здоровье игрока. Здоровье не может стать меньше `0`.

### `void heal(int amount)`

Входные данные:

- `amount` - количество восстанавливаемого здоровья.

Увеличивает здоровье игрока. Здоровье не может стать больше `100`.

### `void addPotionId(int potionId)`

Входные данные:

- `potionId` - ID зелья.

Добавляет зелье в инвентарь игрока.

### `bool removePotionId(int potionId)`

Входные данные:

- `potionId` - ID зелья, которое нужно удалить из инвентаря.

Удаляет первое найденное зелье с указанным ID. Возвращает `true`, если зелье
было найдено и удалено. Возвращает `false`, если такого зелья нет.

### `void clearPotions()`

Очищает инвентарь игрока от всех зелий.

### `std::vector<int> getPotionIds() const`

Возвращает список ID зелий в инвентаре.

### `bool isAlive() const`

Проверяет, жив ли игрок. Возвращает `true`, если здоровье больше `0`.
