# Game

Класс `Game` управляет всей игрой. Он отвечает за главное меню, создание мира,
обработку команд игрока, перемещение между локациями, работу с сундуками,
инвентарём, зельями, сохранением и загрузкой.

## Поля

- `Player player` - объект игрока.
- `std::vector<Location> locations` - список всех локаций игры.
- `std::vector<Potion> potions` - список всех типов зелий.
- `std::vector<Chest> chests` - список всех сундуков.
- `bool gameWon` - состояние победы. Если значение `true`, игра завершается победой.

## Публичные методы

### `void run()`

Запускает игру и показывает главное меню.

## Внутренние методы

### `void showMainMenu() const`

Выводит главное меню игры.

### `void handleMainMenu()`

Считывает выбор пользователя из консоли и запускает новую игру, загрузку или
выход.

### `void startNewGame()`

Создаёт новое состояние игры и запускает игровой цикл.

### `void createLocations()`

Создаёт игровые сущности: зелья, сундуки и локации. Также задаёт связи между
локациями и ASCII-отрисовку.

### `void showCurrentLocation() const`

Выводит информацию о текущей локации игрока: здоровье, название, описание,
ASCII-карту, сундуки и доступные переходы.

### `void gameLoop()`

Основной игровой цикл. Работает, пока игрок жив, не вышел из игры и не достиг
победы.

### `void showActions() const`

Выводит список доступных действий во время игры.

### `void handleCommand(int command)`

Входные данные:

- `command` - номер команды, выбранный игроком.

Выполняет действие игрока: открыть сундук, показать инвентарь, перейти в другую
локацию, выпить зелье, сохранить игру или выйти.

### `void openChest()`

Открывает сундук в текущей локации, если он есть и ещё не был открыт. После
открытия добавляет зелья из сундука в инвентарь игрока.

### `void showInventory() const`

Выводит содержимое инвентаря игрока.

### `void drinkPotion()`

Позволяет игроку выбрать зелье из инвентаря по ID и восстановить здоровье.

### `void saveGame() const`

Сохраняет состояние игры в файл `save.txt`: текущую локацию, здоровье,
инвентарь, открытые локации и открытые сундуки.

### `void loadGame()`

Загружает состояние игры из файла `save.txt`. Если файла нет, создаёт новую
игру.

### `void moveToLocation()`

Считывает ID целевой локации из консоли и пытается переместить туда игрока. Если
локация закрыта, запускает головоломку.

### `Location* findLocationById(int locationId)`

Входные данные:

- `locationId` - ID локации, которую нужно найти.

Ищет локацию по ID. Возвращает указатель на локацию, если она найдена, или
`nullptr`, если такой локации нет.

### `bool solveLocationPuzzle(int locationId)`

Входные данные:

- `locationId` - ID закрытой локации, для которой нужно решить головоломку.

Показывает головоломку для выбранной локации и проверяет ответ игрока.
Возвращает `true`, если ответ правильный.

### `void showConnections(const Location& location) const`

Входные данные:

- `location` - локация, для которой нужно показать доступные переходы.

Выводит список ID локаций, доступных для перехода.

### `void printSeparator() const`

Выводит короткий разделитель в консоль.

### `void printBigSeparator() const`

Выводит большой разделитель в консоль.

### `std::string getPotionInfoById(int potionId) const`

Входные данные:

- `potionId` - ID зелья.

Возвращает строку с названием зелья и количеством восстанавливаемого здоровья.
Если зелье не найдено, возвращает строку `"Unknown potion"`.
